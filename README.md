# 音乐分享网站

一个基于 PHP + MySQL 的音乐分享平台，支持音乐上传、分享、评论和数据可视化管理。

## 功能特点

### 用户功能
- 🎵 浏览随机音乐推荐
- 📱 响应式设计，支持移动端
- 🔗 通过分享链接访问特定音乐
- 💬 发表评论留言（IP限制）
- 🎧 在线音乐播放

### 管理员功能
- 👨‍💼 完整的管理后台
- 📊 数据可视化仪表盘
- 🎵 音乐文件管理（增删改查）
- 💬 评论审核管理
- ⚙️ 系统参数配置
- 📈 数据统计和分析

## 技术栈

- **后端**: PHP 7.4+
- **数据库**: MySQL 5.7+
- **前端**: HTML5, CSS3, JavaScript
- **存储**: 本地文件系统

## 安装指南

### 环境要求
- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- 支持文件上传的Web服务器（Apache/Nginx）
- 启用以下PHP扩展：mysqli, gd, fileinfo
